# Stumap Website Frontend

Modern, responsive and scalable landing page implementation for the Stumap platform.

---

# Tech Stack

## Core Technologies

| Technology | Purpose |
|---|---|
| React | Component-based frontend architecture |
| TypeScript | Type safety and maintainability |
| Vite | Fast development/build tooling |
| TailwindCSS | Utility-first styling |
| React Router DOM | Page routing |
| Framer Motion | Smooth animations |
| Axios | API requests |
| Lucide React | Modern icon system |

---

# Project Goals

The frontend must:

- Match the Figma design pixel-perfect
- Be fully responsive
- Follow reusable component architecture
- Support scalability for future modules
- Maintain clean and readable code structure
- Be optimized for performance

---

# Folder Structure

```bash
src/
│
├── assets/            # Images, icons, illustrations
├── components/        # Reusable UI components
│   ├── common/
│   ├── layout/
│   └── sections/
│
├── pages/             # Route pages
│   └── Home/
│
├── layouts/           # Shared layouts
├── hooks/             # Custom React hooks
├── services/          # Axios/API services
├── constants/         # Static data/constants
├── styles/            # Global styles
├── utils/             # Utility functions
├── types/             # TypeScript types/interfaces
└── router/            # Router configuration
```
